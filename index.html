My Projects

Proxy WhatsApp API using web scraping

Our project aims to create a request-based application API without directly accessing WhatsApp. We have created a proxy API of WhatsApp which can open WhatsApp Web with the help of Firefox Gecko driver, it would scrape the QR code and it takes a screenshot of it and display it in our image gallery and then we can scan it without opening WhatsApp and then we can scan the QR code and then we can log in. it will ask us to enter a contact to integrate the API with. After selecting and integrating the contact, we can send messages, retrieve messages, send images from the directory we specified, and send gif files. We achieved this using Selenium integrated with Python to automate the proxy API. We can use this as an alternative for business API with very low cost
